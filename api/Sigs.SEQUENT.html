Sequent signature used as input to most functors in Cyclist.
type t

"Syntactic" equality. Used to check that closing an open NODE is done via another one marked with a syntactically equal sequent.

As equal but ignoring tags. Used to check that the target of a backlink NODE is equal to that marking it, ignoring tags.

Returns set of tags in sequent.
