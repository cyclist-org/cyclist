val unify_partial : ?order:bool -> ?inverse:bool -> t Sl_unifier.t
Unify all pairs of the 1st argument with a part of the 2nd.
  • The optional argument ~order:false indicates whether the internal ordering of the pair's members is honoured.
  • The optional argument ~inverse:false indicates if the substitution is applied to the second argument (as opposed to the first).
